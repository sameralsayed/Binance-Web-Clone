# 🚀 Binance Web Clone

**Fully functional, sleek & mobile-first cryptocurrency trading web application**

## 📖 Description
A beautiful, responsive recreation of the official Binance app. Buy, sell & trade BTC, ETH, SOL, BNB, PEPE and more with real-time simulated prices, interactive charts, portfolio tracking, instant orders, deposit/withdraw simulation and full wallet experience. Built exactly as requested with pure HTML5, CSS3, JavaScript, jQuery & Bootstrap 5. No folders, no external links.

## ✨ Features
- 📱 100% responsive design (desktop, tablet, mobile phones & all devices)
- 🌙 Dark Binance theme with signature yellow accents (#f0b90b)
- 💰 Live updating prices & 24h change with flash animation
- 📊 Interactive canvas price charts (mini + full modal)
- 🔄 Click any coin → detailed modal with chart & trade form
- ⚡ Quick buy/sell with instant portfolio update
- 🪙 Balance hero + fake pie chart
- 📋 Full markets table (Spot / Futures / P2P tabs)
- 💼 Watchlist, Trending & Holdings sidebar
- 🔍 Working search bar for coins & pairs
- 📲 Mobile bottom navigation (Home / Markets / Trade / Wallet / Earn)
- 💳 Deposit & Withdraw modals
- 🛡️ Security badges (SAFU, 24/7 support) in footer
- 🚀 Fully SEO optimized with rich meta tags

## 🛠️ Technologies
- **HTML5** – Semantic & modern markup
- **CSS3** – Dark theme, animations & Binance branding
- **JavaScript** – Live price engine, canvas charts & state
- **jQuery** – Easy DOM & event handling
- **Bootstrap 5** – Responsive grid, modals & components

## 🚀 How to Run
1. Download the four files (`index.html`, `style.css`, `script.js`, `README.md`)
2. Place them together in one folder (no sub-folders)
3. Open `index.html` in any modern browser
4. Start trading instantly – everything is fully interactive!
